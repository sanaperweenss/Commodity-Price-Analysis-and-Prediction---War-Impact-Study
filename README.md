# Commodity-Price-Analysis-and-Prediction---War-Impact-Study
This project analyzes the impact of war on commodity prices in India, focusing on LPG, crude oil, and essential goods. It uses exploratory data analysis and basic machine learning to identify regional price variations, trends, and the relationship between global oil prices and domestic fuel costs.
## 📌 Project Overview

This project analyzes how global geopolitical events (like war) impact commodity prices in India. It starts with LPG price analysis and expands to include crude oil and other essential commodities, forming a structured, real-world data science pipeline.



# 🚀 Project Phases

## 🟢 Phase 1: Data Collection & Preparation

* Collected LPG price data (city & state level)
* Added crude oil price data
* Structured and cleaned datasets into CSV format

📌 Output:

* `lpg_full_data.csv`
* `oil_prices.csv`


## 🔵 Phase 2: Exploratory Data Analysis (EDA)

* Regional price comparison (city vs state)
* Identified highest & lowest price regions
* Visualized distribution and inequality

📊 Insight:

* Price variation due to logistics and infrastructure

## 🟡 Phase 3: Oil Price Integration

* Merged LPG and crude oil datasets
* Analyzed oil → LPG relationship

📊 Insight:

* LPG prices are influenced by global oil trends



## 🔴 Phase 4: Machine Learning

* Built a basic predictive model
* Features:

  * Location
  * Region type
  * Oil price
  * War indicator

📌 Goal:

* Understand and predict price behavior


## 🟣 Phase 5: Multi-Commodity Expansion

* Added other commodities (food, fuel)
* Unified dataset for comparative analysis

📊 Insight:

* Energy commodities show higher sensitivity to global disruptions
